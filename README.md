# 🧱 TheShope - Aesthetic Tiles Gallery

**TheShope** is a modern, full-stack Tile Gallery application built with **Next.js (App Router)**. It allows users to explore a vast collection of aesthetic tiles, search for their favorites, and manage their personal profiles. The project features a unique design, secure authentication, and a fully responsive layout across all devices.

## 🚀 Live Demo

[View Live Application](https://the-shope-assignment-eight.vercel.app/)

---

## 📖 Project Overview

The purpose of this project is to showcase a curated gallery of tiles with detailed specifications. Users can browse new arrivals, search for specific designs by title, and view high-resolution details of each tile. Secure authentication is integrated via **BetterAuth** to allow users to access private details and manage their profile information dynamically.

### 🔑 Key Features

- **Modern UI/UX**: Crafted with a unique aesthetic using **DaisyUI** and **Tailwind CSS**.
- **Dynamic Tile Gallery**: Browse tiles and use the Hero-UI search bar to find specific designs.
- **Secure Authentication**: Robust Login and Registration system using **BetterAuth** with **MongoDB** adapter.
- **Social Login**: Quick access via **Google Social Authentication**.
- **User Profile Management**: A dedicated profile page where logged-in users can view their data and update their **Name** and **Profile Image URL**.
- **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop views.
- **Interactive Carousels**: Smooth home page banners and sliders implemented with **SwiperJS**.
- **Private Routes**: Secure access control for Tile Details and My Profile pages.
